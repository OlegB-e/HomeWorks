# SHOP.CO — E-Commerce Project

Современный веб-магазин одежды с красивым дизайном и удобным интерфейсом.

## 📁 Структура проекта

```
src/
├── html/              # HTML страницы
│   └── pages/        # Основные страницы
│       └── index.html
│
├── css/              # Стили
│   ├── main.css     # Главный файл (подключает все остальные)
│   ├── base/        # Базовые стили
│   │   ├── reset.css
│   │   ├── fonts.css
│   │   └── variables.css
│   ├── components/  # Компоненты UI
│   │   └── components.css
│   ├── layout/      # Макеты страниц
│   │   └── layout.css
│   ├── sections/    # Стили отдельных секций
│   │   └── sections.css
│   └── utils/       # Утилиты
│       └── animation.css
│
├── js/              # JavaScript код
│   ├── main.js      # Entry point (подключает модули)
│   ├── modules/     # Функциональные модули
│   │   ├── slider.js  # Слайдер отзывов
│   │   ├── promo.js   # Промо-баннер
│   │   └── cart.js    # Корзина
│   └── utils/       # Утилиты и помощники
│
├── assets/          # Статические файлы
│   ├── fonts/       # Шрифты (Satoshi, IntegralCF)
│   └── images/      # Изображения
│       ├── icons/       # SVG иконки
│       ├── logos/       # Логотипы брендов
│       ├── products/    # Фото товаров
│       └── banners/     # Баннеры
│
└── data/            # Данные (JSON)
    ├── products.json    # Товары
    ├── categories.json  # Категории
    └── promo.json       # Промо-коды
```

## 🎨 CSS архитектура

- **main.css** — главный файл, импортирует все остальные в правильном порядке
- **base/** — нормализация, переменные, шрифты (загружаются первыми)
- **components/** — переиспользуемые компоненты (кнопки, карточки, модали)
- **layout/** — макеты страниц (header, footer, grid)
- **sections/** — стили отдельных секций (герой, товары, отзывы)
- **utils/** — анимации, утилиты

## 🔧 Как запустить

1. Откройте `src/html/pages/index.html` в браузере
2. Или используйте Live Server (VS Code):
   - Установите расширение "Live Server"
   - Правый клик на `index.html` → "Open with Live Server"

## 📦 Зависимости

- **Swiper 11** — слайдер отзывов (подключается из CDN)
- **Fonts** — Satoshi, IntegralCF (локально в `/assets/fonts/`)

## 🚀 Функции

- ✅ Адаптивный дизайн
- ✅ Слайдер отзывов (Swiper)
- ✅ Корзина товаров
- ✅ Промо-баннер с закрытием
- ✅ Модальные окна
- ✅ Красивые анимации
